Como instalé admcycles:
Seguí esta guía: https://gitlab.com/modulispaces/admcycles/-/blob/master/README.rst
>Bajé SageMath por medio del link ahí, si hay una nueva versión instalar sobre la anterior y luego correr el pkg para overridear cosas viejas.
>Bajé admcycles con el comando "sage -pip install admcycles"
En VSCode, creamos un archivo ipynb que pide el kernel, entonces escogemos jupyter kernel de sagemath
Luego usar from admcycles import * y gg
